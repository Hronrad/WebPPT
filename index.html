PART 05：模板选择

不同汇报场景的页面骨架

Academic

学术报告

问题背景 → 理论框架 → 公式推导 → 实验/模拟 → 结论。

Teaching

课程教学

概念引入 → 直观动画 → 例题操作 → 误区对照 → 总结。

Research

研究进展

目标 → 当前结果 → 瓶颈 → 下一步计划 → 风险。

Project

产品/项目汇报

痛点 → 方案 → 流程演示 → 指标 → 路线图。

Analytics

数据分析

指标概览 → 筛选对比 → 趋势解释 → 结论建议。

Defense

答辩/立项

核心主张 → 创新点 → 可行性 → 质疑回应 → 收束页。

AI 选型原则

先判断汇报目标，再选择骨架；如果内容涉及模型、参数、流程、概率、对比或推导，应优先插入一页可操作的动态演示，而不是增加纯文字说明。
